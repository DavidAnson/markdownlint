# fenced_code_blocks

Code block with a language specifier not in allowlist:

```html
<h1>markdownlint</h1>
```

Code block with a language specifier not in allowlist:

```css
body {}
```

Code block with a language specifier in allowlist:

```js
console.log('markdownlint')
```

Code block with a language specifier in allowlist:

```scss
body {
  h1 {
    color: red;
  }
}
```
